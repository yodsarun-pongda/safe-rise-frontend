# Pose Monitor (React + Vite)

UI สำหรับดูวิดีโอสตรีม + รับ JSON realtime (WebSocket) เพื่อแสดงผล sit / sleep / stand

## Run
```bash
npm install
npm run dev
```

## ใช้งาน
- ใส่ Video URL (HLS .m3u8 / MJPEG / mp4)
- ใส่ WebSocket URL แล้วกด Connect
- หรือเปิด Mock เพื่อทดลอง UI โดยไม่ต้องมี backend
